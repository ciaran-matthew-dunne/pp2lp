\begin{abstract}
	\noindent
	Within the \bmethod{} ecosystem, the \emph{Predicate Prover} (\pp)
	is an automated theorem prover that discharges first-order
	proof obligations.
	%
	Because \pp's source code is not publicly available,
	its results are currently trusted without independent verification.
	%
	We present \tool, a tool that reconstructs \pp{} proofs in \lp,
	a proof assistant based on the \lpcmr.
	%
	Each of \pp's inference rules is encoded as a \lp{} symbol whose type
	captures the rule's premises and conclusion and whose body \emph{proves}
	the rule sound with respect to classical logic.
	%
	\pp{} can be instrumented to emit \emph{replays} of its proofs;
	\tool{} parses the replay, rebuilds the proof tree,
	and emits a tactic script that \lp{} independently type-checks.
	%
	We prove the soundness of nearly all of \pp's inference rules
	and reconstruct proofs across several (real-world and synthetic)
	benchmark suites.
\end{abstract}
