\begin{abstract}
  Atelier~B's \emph{Predicate Prover} (\pp) is a closed-source automated
  theorem prover that discharges the first-order proof obligations
  generated by \bmethod{} developments, which are widely used to develop
  and certify safety-critical software. Because \pp's source code is not
  publicly available, its verdicts are currently trusted without
  independent verification. We present \tool, a tool that reconstructs
  \pp{} proofs in \lp, a proof assistant based on the \lpcmr. We encode
  each of \pp's inference rules as a \lp{} symbol whose type captures the
  rule's premises and conclusion and whose body \emph{proves} the rule
  sound against a small set of \bbook{} primitives---so the rules are
  checked, not postulated. \pp{} can be instrumented to emit a compact
  proof \emph{trace}, which \ab{} expands into a \emph{replay}; \tool{}
  parses the replay, rebuilds the proof tree, and emits a tactic script
  that \lp{} independently type-checks. \tool{} covers nearly all of
  \pp's inference rules and re-checks proofs across several benchmark
  suites, including the non-arithmetic fragment of \ab's own proof
  corpus.
\end{abstract}
